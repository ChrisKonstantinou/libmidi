# 获取源码

```bash
git clone --recurse-submodules https://github.com/tanluteam/libmidi
```

# 构建

此项目构建系统是基于 [CMake](https://cmake.org/)。

### 构建命令:

```bash
mkdir -p build && cd build
cmake -DCMAKE_BUILD_TYPE=Release .. && cmake --build .
```
